Search syntax

Examples:
  • "china" - searches china in title
  • "china | japan" - searches china, or japan in title
  • "title = china" - searches anything with China in title
  • "tags__tag = tag" - searches links with tag
  • "bookmarked = True" - all bookmarked link
  • "(title = china | title = japan) & bookmarked = True" - all bookmarked link, with china, or japan in title
Available fields:
  • {% for entry_query_name in entry_query_names %} "{{entry_query_name}}", {% endfor %}
Available comparison operators:
  • {% for entry_query_operator in entry_query_operators %} "{{entry_query_operator}}", {% endfor %}
  • "==" for exact compare, "=" to find items containing string, ">" for comaparing if greater than, "<" for comaparing if less than, ">=", "<=", "~" for negation
  • Standard Django operators are also supported. "title__isnull == True" Is correct search term.
Available condition operators
  • "&" - And, "|" - Or, "~" - Not
System accepts brackets.
